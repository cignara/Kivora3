## Auth
No server or Firebase needed. Login = child name + avatar -> localStorage.
Progress saved per child. Works offline. Works on GitHub Pages.

## Stack
Static HTML, vanilla JS, Google Fonts, Browser Web Speech API, localStorage.
No npm, no build step, no server.
